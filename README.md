Modern enterprise calculator application built with HTML, CSS, and JavaScript.

- Scientific Calculator
- Programmer Mode
- Finance Calculator
- Statistics Tools
- Matrix Calculator
- Graphing Calculator
- Unit Converter
- BMI & Date Tools
- Export / Import
- Offline Ready

## Technology
- HTML5
- CSS3
- JavaScript
- Canvas API
- Local Storage
